# СТАЛКЕРЫ.io

Top-down shooter de arena ambientado en la Rusia post-soviética. HTML5 Canvas, sin dependencias externas.

## Personajes

- 🧟 Grigori — Mutante de Chernóbyl (dispara vómito ácido)
- 🤵 Viktor — Oligarca ruso (tira botellas de vodka)
- 🪖 T-72 — Tanque de guerra soviético
- 🐻 Misha — Oso de circo (tira caca)

## Cómo jugar

Abrí `index.html` en un navegador.

- WASD: movimiento
- Mouse: apuntar
- Click izquierdo: disparar
- Espacio / Click derecho: habilidad especial
- ESC: pausa
